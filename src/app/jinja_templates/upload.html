    Instructions on how to use this web-page

    1. Please cough into your phone's microphone or at your screen and record the sound wave is saved as a .wav file
    2. Please ensure the cough lasts between 2-4 sec.
    3.Upload the wave file to the web ppage.
    4. Please ensure cough recoding is done in a relatively quiet environment with little background noise
    5. Click process on the web page.
    The web-page takes the cough sound and parses the mel-frequency and spectogram through our inference engine.
    The inference engine gives the user a score that is the probability of that user having COVID-19
